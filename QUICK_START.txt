❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Backend won't start?
  → Check: python -m app.main
  → Check port 8000 is free
  → Verify .env file exists

Frontend won't load?
  → Check: npm run dev
  → Check port 5173 is free
  → Check VITE_API_URL in .env.local

CORS errors?
  → Backend needs CORS middleware
  → Check SETUP_GUIDE.md for fix
