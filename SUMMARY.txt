🔗 API ENDPOINTS (Backend Expected)
═══════════════════════════════════════════════════════════════════

Frontend calls these endpoints (configure in api.ts):

POST /deploy
  → Submit new project
  ← Returns task_id

GET /projects
  → Get all projects
  ← Returns project array

GET /projects/{id}
  → Get project details
  ← Returns single project

GET /status/{taskId}
  → Get real-time status
  ← Returns status object

GET /health
  → Health check
  ← Returns status
