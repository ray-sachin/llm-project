⚠️ TROUBLESHOOTING
════════════════════════════════════════════════════════════════════

If frontend shows blank page:
  → Check if backend is running (http://localhost:8000/docs)
  → Check browser console (F12) for errors
  → Check .env.local for VITE_API_URL

If backend won't start:
  → Check if port 8000 is in use: netstat -ano | findstr :8000
  → Verify .env file exists with config
  → Try: uvicorn app.main:app --reload (alternative command)

If ports are already in use:
  → Kill process on port 8000: netstat -ano | findstr :8000
  → Then stop that process ID
  → Or change ports in vite.config.ts
